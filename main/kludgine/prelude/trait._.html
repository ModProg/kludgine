Expand description

Allows comparing floating point numbers with approximation.

Uses [approx::AbsDiffEq] with the default epsilon.

Required Methods§

fn approx_eq(&self, other: &Self) -> bool

Returns true if the values are approximately equal. Uses [approx::AbsDiffEq] with the default epsilon.

Provided Methods§

fn approx_ne(&self, other: &Self) -> bool

Returns true if the values are not approximately equal. Uses [approx::AbsDiffEq] with the default epsilon.

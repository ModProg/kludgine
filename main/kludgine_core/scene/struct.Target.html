Expand description

A render target

Fields

scene: Arc<Scene>

The scene to draw into.

clip: Option<SizedRect<u32, Pixels>>

The curent clipping rect. All drawing calls will be clipped to this area.

offset: Option<Vector<f32, Pixels>>

The current offset (translation) of drawing calls.
